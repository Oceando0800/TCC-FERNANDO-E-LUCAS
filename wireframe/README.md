# Wireframe

Protótipo de baixa fidelidade (HTML/CSS) para visualizar a estrutura das telas do sistema.

## Como abrir

1. Abra `wireframe/index.html` no navegador.
2. Use os links para navegar entre as telas.

## Telas incluídas

- Hub do wireframe
- Login
- Cadastro
- Inicio (usuario)
- Mapa (usuario)
- Painel admin
- Perfil
