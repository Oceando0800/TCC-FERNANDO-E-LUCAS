SCDRI — SISTEMA DE CONTROLE DE RESÍDUOS INDUSTRIAIS

Sistema web desenvolvido para cadastro de usuários e envio de denúncias com geolocalização, autenticação e painel administrativo.

Funcionalidades

Usuário

Cadastro e login

Edição e visualização de perfil

Registro de denúncias com título, descrição, imagem, localização no mapa e nível de urgência

Administrador

Acesso restrito

Visualização e gerenciamento de denúncias

Tecnologias Utilizadas

Front-end

HTML5

CSS3

JavaScript

Leaflet.js

Back-end

Node.js

Express

CORS

Helmet
